### Backend Development 
Backend development is the part of the web or app development that focuses on what happens behind the scenes. It involves building and maintaining the server, database, and application logic that power the features uses interact with on the frontend.

In simple terms:<br>
üëâ Backend development is responsible for how an application works, not how it looks.

### Different HTTPS Request Methods
1. GET Request: GET is an HTTP method used to retrieve or read data from the server. It does not change anything on the server. <br>
Simple meaning: Give me the inormation.

2. POST Request: POST is an HTTP method used to send data to the server to create a new resource. <br>
Simple meaning: Create something new with this information.

3. PUT Request: PUT is an HTTP method used to update or replace an existing resource completely new data. <br>
Simple meaning: Replace the old data entirely with this new data.

4. PATCH Request: PATCH is an HTTP method used to partially update an existing resource. You change only the fields you want. <br>
Simple meaning: Update only this part

5. DELETE Request: DELETE is an HTTP method used to remove or delete a resource from the server. <br>
Simple meaning: Remove this


### ‚≠êShort Summary in One Line Each
- GET -> Read data
- POST -> Create new data
- PUT -> Replace existing data completely
- PATCH -> Update a small part of existing data
- DELETE -> Remove data
