### Different HTTPS Request Methods
1. GET Request: GET is an HTTP method used to retrieve or read data from the server. It does not change anything on the server.
Simple meaning: Give me the inormation.

2. POST Request: POST is an HTTP method used to send data to the server to create a new resource.
Simple meaning: Create something new with this information.

3. PUT Request: PUT is an HTTP method used to update or replace an existing resource completely new data.
Simple meaning: Replace the old data entirely with this new data.

4. PATCH Request: PATCH is an HTTP method used to partially update an existing resource. You change only the fields you want.
Simple meaning: Update only this part

5. DELETE Request: DELETE is an HTTP method used to remove or delete a resource from the server.
Simple meaning: Remove this
